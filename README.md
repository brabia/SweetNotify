# SweetNotify
sNotify
==========
sNotify is a Sweet Notify based on javascript

Usage
-----

```html
<script src="sNotify/js/sNotify.min.js"></script>
<link rel="stylesheet" type="text/css" href="sNotify/css/sNotify.css">
```

Tutorial
--------

Examples
--------

```javascript
sNotify({
	title: "Are you sure?",
	text: "This is sNotify content!",
	type: "warning"
}, function(){
	
});
```
